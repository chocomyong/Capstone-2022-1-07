# 1. 프로젝트 소개
* 프로젝트 이름 : Super Resolution을 이용한 갤러리 애플리케이션
* 프로젝트 목적 : 갤러리의 기본기능을 제공하고 초해상화 기능을 이용하여 저화질 이미지를 고화질 이미지로 변환한다.
* 프로젝트 개요 : 




본 프로젝트는 개인의 안면 사진을 이용하여 퍼스널 컬러와 성별을 판별해 주는 프로젝트입니다.
